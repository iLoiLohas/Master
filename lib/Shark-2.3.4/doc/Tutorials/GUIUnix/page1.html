Compilation of GUI Examples

This tutorial will explain how to compile the GUI example programs which come with the ReClaM module under Unix-based systems.

Prerequisites

It is assumed that the Shark library is properly installed on your system. For help on this topic you can consult the Installation Tutorial. Depending on your installation, it may be necessary to set the LD_LIBRARY_PATH (or DYLD_LIBRARY_PATH under MacOS) environment variable to the directory where the Shark library is installed.

For all GUI examples, Qt is required. It is assumed that qmake is installed correctly.

For some of the GUI examples, additionally Qwt is needed. Depending on your installation, it may be necessary to set the LD_LIBRARY_PATH (or DYLD_LIBRARY_PATH) environment variable to the directory where the Qwt library is installed.

The GUI Example Directory

The ReClaM GUI example programs are located in the examples/ReClaM/QT/ directory under the Shark root directory [SharkRoot] (i.e., where you extracted the library).
