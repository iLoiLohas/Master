Building the Examples

After the corresponding project file is adapted, the example program, say classification, can be simply build by:

> cd [SharkRoot]/examples/ReClaM/QT > qmake classification.pro > make

If you get the error message

./classification: error while loading shared libraries: libShark.so.2: cannot open shared object file: No such file or directory

you have to set the LD_LIBRARY_PATH (or DYLD_LIBRARY_PATH under MacOS) environment variable to the directory where the Shark library is installed. A similar problem, which is solved analogously, occurs if the program requires Qwt and you did not set the path to the Qwt library correctly,
