My First Arrays

Arrays are basic dynamical data structures in Shark. They are generic containers implemented using templates. Arrays are most often used to handle numerical vectors/matrices/tensors. They are resizable, parameter passing by value and assignment work properly, and range checks can be conducted at run-time.

The Array class provides a lot of functions, which are described in the Array documentation. Only the most basic concepts are introduced in the following tutorial.

In the first example, we define three Arrays. 

#include <Array/Array.h>

int main(int argc, char** argv)
{
    Array<double> a, b, c;
    a.resize(3u);
    b.resize(3u);

    std::cout << a.ndim() << " " << a.dim(0) << std:endl;

    a = 1.;
    for(unsigned i=0; i<b.dim(0); i++) b(i)=(i+1)/10.;

    return EXIT_SUCCESS;
}

The Arrays a and b are resized to be one-dimensional and to have three elements. The member functions ndim() and dim(n) return the number of dimensions and the number of elements in dimension n, respectively. That is, the output is:

1 3

Then every element of a is set to 1, i.e., a corresponds to the unit vector. The vector b is filled with the values 0.1, 0.2, and 0.3.

Note that round brackets are the typical way to index an Array element.
