SunOS/Linux/MacOS Installation from the Repository

Usually you want to install a standard stable Shark distribution. This is explained here.

However, if you want to install a Shark snapshot from the repository, you should first read the standard installation tutorial and then call installAdmin instead of installShark:

> cd [SharkRoot] > ./installAdmin

The script installAdmin looks like this: 

#!/bin/sh

make -f Makefile-SharkConfig
./SharkConfig
make all
./makedoc
./makerights

In addition to the commands carried out by the standard installation script, installAdmin generates ".in" files for configure first, executes autoconf and ./configure and generates the documentation using doxygen. A proper installation of current versions of autoconf and doxygen as well as LaTeX is required.
