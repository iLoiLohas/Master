Introduction to Sharks random number generator

This tutorial introduces the random number generator implemented in Shark and explain how it is used. The C++ library Rng provides seven discrete and seven continuous probability distributions. There are two ways to use a random number generator. Either be explicitly creating a random number generator of the desired type or by using the class Rng that already contains an instance of a random number generator for several often used probability distributions. The next part will give an example for the second method, followed by an example of the first.

The library contains the following types of random number generators:

Discrete Random Number Generators:

  • Bernoulli
  • Binomial
  • Geometric
  • DiffGeometric
  • HyperGeometric
  • DiscreteUniform
  • Poisson

Continuous Random Number Generators:

  • Cauchy
  • Erlang
  • LogNormal
  • NegExponential
  • Normal
  • Uniform
  • Weibull
