Compilation of GUI Examples

This tutorial will explain how to compile the GUI example programs which come with the ReClaM module under Windows.

The installation procedure may differ depending on which version of Microsoft Windows and which compiler is used. We tested Visual C++ Express 2005 under Windows 2000 and Visual C++ Express 2008 under Windows Vista.

Prerequisites

It is assumed that the Shark library is already built as described in the Installation Tutorial.

For all GUI examples, Qt4 is required (note: in this tutorial the commercial version 4.3.3 is assumed).

For some of the GUI examples, additionally Qwt 5.0.2 is needed.

The GUI Example Directory

The ReClaM GUI example programs are located in the examples/ReClaM/QT/ directory under the Shark root directory [SharkRoot] (i.e., where you extracted the library).
