Prepare Own Programs

To use Shark in your programs you need to make the header files and the library known to the compiler and linker, respectively. Both settings are done in the project settings dialog. This tutorial provides screenshots of Developer Studio 6.0. In Developer Studio .net the same settings may be located in slightly different places in the project settings dialog.

Include Path

Open the tab C/C++ in the project properties dialog and add [shark-root]\include as an additional include path.

Library Path and Name

Then switch to the Link tab and add the library name Shark.lib to the list of libraries used. Set [shark-root]\lib\WinNT\Release or [shark-root]\lib\WinNT\Debug as an additional library path.
