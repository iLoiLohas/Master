Compilation of an Example Program

This tutorial will explain how to compile the example programs which come with the Shark library.

Prerequisites

It is assumed that the Shark library is properly installed on your system. For help on this topic you can consult the Installation Tutorial. Note that you should call make install (or something like make libs installlibs installheaders ) before you can call make examples in the Shark root directory.

The Example Program

All example programs are located in the "examples" directory under the Shark root directory (which is where you extracted the library). This directory contains one sub-directory for each Shark module. For example, let us have a look at the LinAlg examples:

cblnsrch_test.cpp eigensymmJacobi_test.cpp rank_decomp_test.cpp covar_corrcoef_test.cpp eigensymm_test.cpp rank_test.cpp detsymm_test.cpp fft_test.cpp svd_test.cpp dlinmin_test.cpp g_inverse_matrix.cpp svdrank_test.cpp eigenerr_test.cpp linalg_simple_test.cpp svdsort_test.cpp eigensort_test.cpp linmin_test.cpp eigensymmJacobi2_test.cpp lnsrch_test.cpp

Each *.cpp file in this folder is a separate example program. That is, there are 19 examples demonstrating the usage of the LinAlg module. The "g_inverse_matrix" example computes the (generalized) inverse of a 3x3-matrix, see g_inverse_matrix.cpp: 

#include <iostream>
#include "Array/ArrayIo.h"
#include "LinAlg/LinAlg.h"

using namespace std;

// Values used for initialization of input matrix:
double values[9] = {
                       1.0, 2.0, -1.0,
                       2.0, 5.0, -1.0,
                       1.0, 2.0,  0.0
                   };

int main()
{
    unsigned          i, j;      // Counter variables
    Array< double > A(3, 3),   // Input matrix
    B(3, 3);   // Generalized inverse matrix

    // Initialize matrices:
    for (i = 0; i < 3; i++) {
        for (j = 0; j < 3; j++) {
            A(i, j) = values[ i * 3 + j ];
        }
    }
    B = .0;

    // Output of input matrix:
    cout << "input matrix:" << endl;
    writeArray(A, cout);

    // Calculate generalized inverse matrix:
    g_inverse(A, B);

    // Output of generalized inverse matrix:
    cout << "generalized inverse matrix:" << endl;
    writeArray(B, cout);
}
