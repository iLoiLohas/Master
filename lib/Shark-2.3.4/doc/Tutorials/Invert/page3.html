Advanced Options

The g_inverse method has some optional parameters, which usually need no tunig by the user: 

#include <LinAlg/LinAlg.h>
[...]

  unsigned maxIterations = 75
  double tolerance = 1e-10;
  bool ignoreException = true;

  g_inverse(matrix, inverse, maxIterations, tolerance, ignoreException);

[...]
The method is based on a singular value decomposition (SVD). The variable maxIterations defines the maximum number of iterations the algorithms spends for diagonalization of the bidiagonal form in the SVD.

All singular values below the threshold tolerance will be set to zero.

If maxIterations iterations are not enough for the diagonalization to converge, the SVD method throws a SharkException. If ignoreException is true this exception is ignored by g_inverse. The default is ignoreException = false.
