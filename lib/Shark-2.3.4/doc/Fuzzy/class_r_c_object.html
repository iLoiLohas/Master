Detailed Description

Base class for Reference Counted Objects.

This base class contains functionality that enables the derived classes to be handled by Refernce Counted Pointers (as described in Scott Meyers, More Effective C++). The explicit use of this methods should not be necessary, they should only be used by RCPtr objects.
