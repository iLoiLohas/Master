Detailed Description

FuzzySet with trapezoid membership function.

This class implements a FuzzySet with a trapezoid membership function. A trapezoid membership function is definded by four values a,b,c,d. Points smaller than a and bigger than d have the value 0. The function increases constantly to the value 1 between a and b, stays 1 between b and c, and decreases constantly between c and d.
