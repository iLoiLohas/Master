Detailed Description

A rule which is composed of premise and conclusion.

The premise is composed of linguistic terms which are linked with one connective (AND, OR, PROD or PROBOR). AND is the minimum function, OR the maximum function, PROD the product, and PROBOR(x,y)= x+y-xy. There is always used solely one kind of connective for all connections in one permise.

Constructor for rules given by a string of the form 

  IF LinguisticVariable1 = LinguisticTermA AND/OR
     LinguisticVariable2 = LinguisticTermB AND/OR ...
  THEN
     LinguisticVariable3 = LinguisticTermC
