Detailed Description

A composed LinguisticTerm.

A composed LinguisticTerm makes it possible to do some calculations on fuzzy sets, e.g. to connect a constant fuzzy set and a sigmoid fuzzy set using the minimum fuction. This results in a sigmoid fuzzy set which is cutted at the value of the constant fuzzy set.
