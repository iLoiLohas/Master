Detailed Description

template<class T>
class SearchAlgorithm< T >

Search algorithm template with flexible search space.

The SearchAlgorithm template class represents an abstract search algorithm for search on the space represented by the template type T. The search is guided by a single- or multi- objective fitness function, defined on this space.
