Detailed Description

"Principal Component Analysis" class for data compression.

The Principal Component Analysis also known as Karhunen-Loeve-Transformation takes a symmetric matrix and uses its decomposition

where is the diagonal matrix of eigenvalues of and is the orthogonal matrix with the corresponding eigenvectors as columns. then defines a successive orthogonal rotation, that maximizes the variances of the coordinates, i.e. the coordinate system is rotated in such a way that the correlation between the new axes becomes zero. If there are axes, the first axis is rotated in a way that the points on the new axis have maximum variance. Then the remaining axes are rotated such that a another axis covers a maximum part of the rest variance, that is not covered by the first axis. After the rotation of axes, the rotation destination of axis no. is fixed. An application for PCA is the reduction of dimensions by skipping the components with the least corresponding eigenvalues/variances.
