Detailed Description

Meta model (training scheme) for on-line Crammer-Singer multiclass learning.

Author:
M. Tuma
Date:
2010
This algorithm implements an epoch-based approach to solving the optimization problem occuring in a Crammer-Singer multi-class machine.
One epoch is understood as the solver in a perceptron-like manner looking at all training examples once, possibly carrying out intermediate optimization steps on variables corresponding to known examples in-between.
Examples:

McSvm.cpp.
