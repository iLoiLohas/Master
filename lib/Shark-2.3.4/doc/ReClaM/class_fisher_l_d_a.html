Detailed Description

Fisher's Linear Discriminant Analysis for data compression.

Similar to PCA, Fisher's Linear Discriminant Analysis is a method for reducing the datas dimensionality. In contrast to PCA it also uses class information.

Consider the datas covariance matrix and a unit vector which defines a one-dimensional subspace of the data. Then, PCA would maximmize the objective , namely the datas variance in the subspace. Fisher-LDA, however, maximizes

where is the covariance matrix of the class-means and is the average covariance matrix of all classes (in both cases, each class' influence is weighted by it's size). As a result, Fisher-LDA finds a subspace in which the class means are wide-spread while (in average) the variance of each class becomes small. This leads to good lower-dimensional representations of the data in cases where the classes are linearly separable.

If a subspace with more than one dimension is requested, the above step is executed consecutively to find the next optimal subspace-dimension orthogonally to the others.

Note: the max. dimensionality for the subspace is #NumOfClasses-1.

For more detailed information about Fisher-LDA, see Bishop, Pattern Recognition and Machine Learning.
