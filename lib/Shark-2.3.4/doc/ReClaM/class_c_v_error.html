Detailed Description

ErrorFunction based on a cross validation procedure

The CVError class computes the mean error over partitions, that is, the cross validation error. It trains the sub-models defined by the CVModel object with a given Optimizer on a given ErrorFunction using the training parts of the partitions. For the mean error computation, it uses the varidation part of the partitions, which contain every example exactly once.
