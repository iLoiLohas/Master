Detailed Description

Quadratic program solver for multi class SVMs with sum-to-zero constraint.

The solver solves the same type of problem as QpMcDecomp, but with the additional constraint for the variables corresponding to each training example .

unsigned int  vi,
unsigned int  vj 
) const [inline, protected]

Return the 'kernel modifier' which is used to compute the Q-matrix from the kernel matrix.

Let (i, m) and (j, n) denote the variables to compare, then the Q-matrix entry takes the form , where M denotes the so-called modifier. In general, it depends on four label-valued terms. However, due to symmetry it depends only on the six possible binary equality relations among these, resulting in different values.

Parameters:
yilabel of the example corresponding to the first variable (i,m)
yjlabel of the example corresponding to the second variable (j,n)
vilabel 'm' of the first variable
vjlabel 'n' of the second variable
