Detailed Description

Precomputed Kernel Gram matrix.

The PrecomputedKernelMatrix class may be beneficial for certain model selection strategies, in particular if the kernel is fixed and the regularization parameter is varied.
This class is a memory intensive alternative to the plain KernelMatrix. It should be used only for problem sizes of up to a few thousand examples. This class is not yet well integrated with the CachedMatrix class, such that data is redundantly.
The SVM_Optimizer class does not support this option yet.
