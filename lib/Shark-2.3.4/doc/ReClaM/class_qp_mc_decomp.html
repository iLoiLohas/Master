Detailed Description

Quadratic program solver for box constrained multi class problems.

This algorithm solves the same problem as QpBoxDecomp, with the difference that the structure of the quadratic program of multi class classification is exploited for more efficient caching and a more natural problem encoding.

Return the 'kernel modifier' which is used to compute the Q-matrix from the kernel matrix.

Let (i, m) and (j, n) denote the variables to compare, then the Q-matrix entry takes the form , where M denotes the so-called modifier. In general, it depends on four label-valued terms. However, due to symmetry it depends only on the six possible binary equality relations among these, resulting in different values.

Parameters:
