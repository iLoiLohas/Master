Variable Documentation

unsigned i

Initializes some internal variables used by the BFGS algorithm, calculates the current error of the model.

The given parameters are used to initialize some variables used by the BFGS and the line search algorithms and to allocate memory for internal structures. The current error value of the used model is determined.

Parameters:
inInput patterns for the currently used model.
targetTarget values corresponding to the input patterns.
fretInitial error of the currently used model.
lsType of LineSearch algorithm, that is used by the BFGS algorithm. Possible values: "Dlinmin" (default value), "Linmin", "Cblnsrch"
axInitial left bracket for the line search algorithms, the default bracket is "0".
bxInitial right bracket for the line search algorithms, the default bracket is "1".
lambdaThe initial value for the searched , that will minimize the function used by the line search algorithm. The default value is "0.25".
Returns:
none
