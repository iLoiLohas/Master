Detailed Description

Epoch-based quadratic program approximator for Crammer-Singer type multi-class machines.

This algorithm implements an epoch-based approach to solving the optimization problem occuring in a Crammer-Singer multi-class machine.
One epoch is understood as the solver in a perceptron-like manner looking at all training examples once, possibly carrying out intermediate optimization steps on variables corresponding to known examples in-between.
Note the following conventions that are used in the documentation and underlie the code: A destinction is made between support vectors (SVs), support classes (SCs), and support patterns (SPs). An SV is a combined pair of input pattern i and potential class label c for which the corresponding coefficient is non-zero. An SC is any label c that makes an input pattern an SV. An SP is any pattern that has at least one SC.
