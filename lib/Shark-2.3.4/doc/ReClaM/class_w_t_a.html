double WTA::error ( Model &  model,
const Array< double > &  input,
const Array< double > &  target 
) [inline, virtual]

Caculates the error based on "the winner takes it all" strategy.

This error measure is used for classification problems. For each pattern only the output neuron with the best (maximum) value is considered and set to 1, all other output neurons are set to zero. This result is compared with the target vectors and then the number of different vectors (= errors) is counted. Finally, the number of errors is normalized with the number of patterns.

Parameters:
modelThe model used for prediction
inputInput vector for the model.
targetTarget vector.
Returns:
The number of errors, normalized with the number of patterns.
