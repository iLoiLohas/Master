Detailed Description

Quadratic program solver for box constrained problems.

This algorithm solves the same problem as QpSvmDecomp but without the equality constraint. For example, this problem corresponds to SVMs without bias. Similar techniques are applied, but this solver solves more complicated sub-problems.
