Detailed Description

Optimize by trying out predefined configurations.

The PointSearch class is similair to the GridSearch class by the property that it optimizes a model in a single pass just trying out a predefined number of parameter configurations. The main difference is that every parameter configuration has to be explicitly defined. It is not possible to define a set of values for every axis; refer to GridSearch for this purpose. Thus, the optPointSearch class allows for more flexibility.
