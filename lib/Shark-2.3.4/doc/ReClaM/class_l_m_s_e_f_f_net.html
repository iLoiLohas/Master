Detailed Description

Offers the functions to create and to work with a feed-forward network with explicit defined neuron-to-bias connections. The network is combined with the mean squared error measure. This combination is created due to computational efficiency.

When you are using the classes FFNet or MSEFFNet it is assumed, that each neuron is connected with the (B)ias value. These classes are a specialization of this class, where you have the free choice to connect each neuron to the bias value or not.
