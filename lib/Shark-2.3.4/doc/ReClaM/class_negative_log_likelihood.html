Detailed Description

negative logarithm of the likelihood of a probabilistic binary classification model

This class makes two assumptions: First, the data must be a binary classification problem using labels +1 and -1. Second, the model must output values in the unit interval [0, 1] which are interpreted as the probability of an example being positive.
