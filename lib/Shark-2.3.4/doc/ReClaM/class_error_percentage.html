Detailed Description

Calculates the error percentage based on the mean squared error.

Measures the euklidian distance between the model output model(in), calculated from the input vector input, and the target vector target. Besides normalizing the squared error to the number P of patterns, it is also divided with the number of outputs N and the range of the target vectors. The resulting value is used to determine the percentage of errors.

Parameters:
modelThe model used for prediction
inputInput vector for the model.
targetTarget vector.
Returns:
The error percentage E.
