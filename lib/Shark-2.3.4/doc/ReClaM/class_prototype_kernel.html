kernel based on prototype vectors

The PrototypeKernel class provides a kernel function on an arbitrary finite set. The set is (w.r.o.g.) assumed to be given as the first N natural numbers {0, ..., N-1}. There is one prototype vector for each element. Then the kernel is computed as .
