Detailed Description

Meta Model for SVM training.

Author:
T. Glasmachers
Date:
2007
The -SVM is a training scheme for Support Vector Machines for regression. It defines a positive constant controlling loss accuracy and another positive constant C controlling the solution complexity.
The Epsilon_SVM class is a meta model which is based on an SVM model. Its parameter vector is composed of the SVM hyper- parameters, that is the hyperparameters C and , and the parameters of a KernelFunction object. As a meta model it does not make sense to call the model or modelDerivative members of this class.
The parameters C and can be stored in a way allowing for unconstrained optimization, that is, the exponential function is used to compute the value from the model parameter. As a side effect, this parameterization is more natural and therefore probably better suited for optimization.
Examples:

SVMregression-gnuplot.cpp, and SVMregression.cpp.
