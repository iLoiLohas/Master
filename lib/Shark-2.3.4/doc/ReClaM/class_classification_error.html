Detailed Description

The ClassificationError class returns the number of classification errors.

This measure is not differentiable.

Examples:

CrossValidation.cpp, KM.cpp, KNN.cpp, simpleRBFNet.cpp, SvmApproximationExample.cpp, SVMclassification-gnuplot.cpp, and SVMclassification.cpp.
