Detailed Description

Noisy version of the chessboard classification problem.

The n-dimensional noisy chessboard problem of size d is defined as a sum of radial Gaussian distributions centered on a grid in . All Gaussians have equal weights and neighboring grid points correspond to opposite class labels.
