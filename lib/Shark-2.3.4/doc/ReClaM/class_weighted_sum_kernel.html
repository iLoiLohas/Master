Detailed Description

Weighted sum of kernel functions.

For a set of positive definite kernels with positive coeffitients the sum

is again a positive definite kernel function. Internally, the weights are represented as to allow for unconstrained optimization. Further, one parameter is removed by fixing the sum of the weights to .
