void ArrayBase::resize ( unsigned  i,
bool  copy = false 
) [inline, inherited]

Resizes an array to an one-dimensional array of size "i".

When the copy flag is set, then existing elements are copied (if possible), otherwise all existing elements are discarded.

Parameters:
ithe new size of the one-dimensional array.
copyif set to "true", then existing elements are copied, otherwise they are discarded.
Returns:
none
Exceptions:
SharkExceptionthe type of the exception will be "size mismatch" and indicates, that you've tried to resize a static array reference and the new size is not equal to the old size

void ArrayBase::resize ( const ArrayBase &  v,
bool  copy = false 
) [inline, inherited]

Resizes an array to the dimensions of array "v".

The new dimensions of the current array are adopted from the dimensions of array v.
When the copy flag is set, existing elements are copied (if possible), otherwise all existing elements are discarded.

Parameters:
vthe array which dimensions are taken as new dimensions of the current array.
copyif set to "true", then existing elements of the current array are copied, otherwise they are discarded.
Returns:
none
Exceptions:
SharkExceptionthe type of the exception will be "size mismatch" and indicates, that you've tried to resize a static array reference and the new size is not equal to the old size
