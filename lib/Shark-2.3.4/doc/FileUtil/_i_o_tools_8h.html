Used to read input patterns and their corresponding target values from a file.

The data for input patterns and the corresponding target values is read from a file and stored in two arrays, that can be used as direct input for networks.
Each line of the file will contain the values for one input pattern, followed by the corresponding target values for this pattern. The single values must be separated by whitespaces.

Parameters:
filenameThe name of the file, from which the data is read.
idThe input dimension, i.e. the no. of input neurons.
odThe output dimension, i.e. the no. of output neurons.
inThe input patterns will be stored here.
outThe target patterns will be stored here.
Returns:
none.
Example
  #include "Array/ArrayIo.h"
  #include "FileUtil/IOTools.h"

  void main()
  {
      Array< double > input, target;

      loadOrganizedData( "test.dat", 4, 2, input, target );

      cout << "input patterns:" << endl;
      writeArray( input, cout );
      cout << "\ntarget values:" << endl;
      writeArray( target, cout );
  }
