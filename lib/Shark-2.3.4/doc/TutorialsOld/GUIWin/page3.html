Building the Examples

After the corresponding project file is adapted, start the Qt Command Prompt from Startmenu (e.g. Start -> Programs -> Qt by Trolltech v4.3.3 (Eval. VS2005) -> Qt 4.3.3 Command Prompt).

Now, a Visual C++ project file for an example program, say classification, can be build by:

> cd [SharkRoot]\examples\ReClaM\QT > qmake -t vcapp classification.pro

This builds classification.vcproj which can be opened in Visual Studio and built there.

If you get an error like fatal error C1083: Cannot open include file: 'ReClaM/Dataset.h': No such file or directory, you should check again if the qmake project file is prepared with the right SHARKHOME variable.

To run the generated .exe-file, the program must find the Qt DLL files (and qwt5.dll, depending on the example). This can be achieved in several ways, for instance by copying the needed DLL files in the example directory or by setting the Windows PATH environment variable accordingly.
For the examples not using qwt it is sufficient to start them from the Qt Command Prompt:

> cd [SharkRoot]\examples\ReClaM\QT > classification
