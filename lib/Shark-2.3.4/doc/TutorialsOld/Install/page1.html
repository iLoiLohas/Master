Welcome to the installation tutorial!

This tutorial explains how to install the Shark machine learning library on Unix systems including Linux and Mac OS.

Intended Audience

The tutorial is designed for users of the Shark library who just downloaded the sources and need help for the proper installation.

Prerequisites

The installation can be carried out either as a user or as an administrator/root user.

The size of the fully installed library varies depending on your platform and your needs. In doubt, be sure to have at least 100 MB of disk space available for the installation.

As the Shark library is distributed as C++ source code you need a C++ compiler for the installation. Because Shark makes full use of modern language features like templates and exceptions be sure that your compiler is up to date. The GNU compilers starting from version 3 should not cause problems.
