Using GlobalRng.h

The class Rng contains instances of several often used discrete and continuous random number generators. The following example programs explains how to use them.

First include the header file containing the Rng class: 

#include <GlobalRng.h>

Next initialize the seed shared by all randm number generators in the class Rng: 

int main(){
   Rng::seed(12);
   return 0;
}

The instance of a Bernoulli random generator is called coinToss

, with p as the (optional) probability for the occurrence of the event. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   double p=0.5;
   std::cout << "Bernoulli trial: " << Rng::coinToss(p) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Bernoulli trial: 0

discrete simulates a uniform distribution with integer numbers, where low and high are the minimal and maximal possible value. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   int low=0, high=10;
   std::cout << "Discrete distribution number: "
             << Rng::discrete(low, high) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Discrete distribution number: 2

A (continuously) uniformly distributed random number generator is named uni (Again low and high are the minimal and maximal possible value.) 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   double low_uni=0., high_uni=10.;
   std::cout << "Uniform distribution number: "
             << Rng::uni(low_uni, high_uni) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Uniform distribution number: 0.794262

A gaussian (normal) random number generator is named Normal (Its parameters are mean and variance of the distribution. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   double mean=0., variance=1.;
   std::cout << "Normal distribution number: "
             << Rng::gauss(mean, variance) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Normal distribution number: -1.05016

An instance of a simulated standard Cauchy distribution is called cauchy. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   std::cout << "Cauchy distribution number: "
             << Rng::cauchy() << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Cauchy distribution number: 0.293598

An instance of a class simulating a differntial geometric distribution is called diffGeom, with p_diffGeom as the probability for a single Bernoulli trial. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   double p_diffGeom=0.1;
   std::cout << "Differential Geometric distribution number: "
             << Rng::diffGeom(p_diffGeom) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Differential Geometric distribution number: -2

A poisson distributed random number generator is called poisson, with lambda as its mean. 

#include <GlobalRng.h>
#include <iostream>

int main(){
   Rng::seed(12);


   double lambda=0.5;
   std::cout << "Poisson distribution number: "
             << Rng::poisson(lambda) << std::endl;


   return 0;
}

results (for example) in:

> ./rngtest Poisson distribution number: 13
