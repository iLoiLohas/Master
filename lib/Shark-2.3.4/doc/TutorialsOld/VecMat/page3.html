Higher Matrix Operations

On matrices, lots of non-trivial operations are defined that require specialized and robust algorithms. These include inversion, eigen-decomposition, and the computation of the determinant. Other operations like transposition and trace are easier to implement but equally important. The Shark Matrix class provides the following operations as member functions:

  • Matrix Matrix::transpose() const
  • Matrix Matrix::inverse() const
  • Matrix Matrix::inverseSymm() const
  • void Matrix::svd(Matrix& U, Matrix& V, Vector& lambda)
  • void Matrix::eigensymm(Matrix& eigenvectors, Vector& eigenvalues) const
  • void Matrix::eigenvalues(Vector& real, Vector* imaginary = NULL) const
  • double Matrix::detSymm() const
  • double Matrix::logDetSymm() const
  • double Matrix::trace() const
Most of these operations are implemented as frontends to the main functionality of the LinAlg module.

Power Series

One of the most important functions in mathematics, the exponential function, is available for square matrices. 

    Matrix t(3, 3, true);       // t is the zero matrix
    Matrix g = exp(t);          // g is the unit matrix

The exponential function is computed by means of its Taylor series. It is easy to use the powerseries functionality to compute other analytic functions on matrices.
