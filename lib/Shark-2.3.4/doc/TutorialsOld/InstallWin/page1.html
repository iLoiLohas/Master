Welcome to the installation tutorial!

This tutorial guides you through the installation of the Shark machine learning library on a system running Microsoft Windows with Developer Studio version 6.0 or higher. If you plan to use Developer Studio 6.0 be sure to have all service packs installed. Shark compiles fine using Visual C++ Express 2005 and 2008.

Download Archives

If you have not done so yet, download the latest release of the Shark machine learning library from Sourceforge.net.

Extract the Archives

Extract the archive Shark-x.x.x.tgz (where x.x.x is the version number) using any tool that understands .tgz format (gzipped tar archive), for example winace. The extracted files are all located in the single folder Shark-x.x.x. This folder will be referred to as [SharkRoot] in the following.
