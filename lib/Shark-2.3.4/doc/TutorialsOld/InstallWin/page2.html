The Workspace

Enter the [shark-root] folder. There you find two visual studio workspace files. These correspond to the two possibilities to install the Shark library:

  • Open Shark_Libs.dsw to create the static Shark library only.
  • Open Shark_All.dsw to create the static Shark library and compile the example programs.

If the workspace is opened with a version of Visual Studio .net the workspace and project files need to be converted to the new format first:

The workspace Shark_Libs.dsw contains a single project for the library. In contrast, Shark_All.dsw contains an additional project for each example program shipped with the library.
