If you have installed Shark with the example programs you can skip this page.

Example Program Workspace

For each Shark module there is one workspace file (*.dsw) located in the corresponding examples sub-directory. Open the file [SharkRoot]\examples\LinAlg-Examples.dsw with Microsoft Developer Studio version 6.0 or later. If you have a ".net" version you are asked to convert workspace and project files to a more recent format:

Just agree to such questions. The workspace contains all examples demonstrating the LinAlg module. Chose Build/Batch Build from the menu to open the batch build dialog.

In Visual Studio 6.0 press Rebuild All:

If you are using some later version of Visual Studio you will have to first Select All and then Rebuild:

This should compile and link all example programs of the LinAlg package. The executables are placed in the folders

  • [SharkRoot]\examples\LinAlg\WinNT\Release
  • [SharkRoot]\examples\LinAlg\WinNT\Debug
