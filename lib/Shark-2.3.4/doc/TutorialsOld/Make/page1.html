Simple Linux Makefile

A very basic Linux Makefile for building a program using Shark may look like this: 

SHARKHOME=/opt/Shark/

LDLIBS=-lshark
LDFLAGS=-L${SHARKHOME}/lib -Wl,-rpath,${SHARKHOME}/lib
CXXFLAGS=-O3 -I${SHARKHOME}/include

The variable SHARKHOME must be set by the user to where Shark was installed.

Now, to compile the source code MySharkExample.cpp just type make MySharkExample:

> make MySharkExample
g++ -I/opt/Shark/include -L/opt/Shark/lib -Wl,-rpath,/opt/Shark/lib MySharkExample.cpp -lshark -o MySharkExample
