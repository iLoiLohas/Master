Most Simple Makefile

When using g++ under SunOS/Solaris, you can specify the runpath using -R. So, a very basic Makefile for building a program that uses Shark with g++ under SunOS/Solaris looks like this: 


SHARKHOME=/opt/Shark/

LDLIBS=-lshark
LDFLAGS=-L${SHARKHOME}/lib -R${SHARKHOME}/lib
CXXFLAGS=-O3 -I${SHARKHOME}/include

The variable SHARKHOME must be set to where Shark was installed. Now, to compile the program MySharkExample.cpp just type make MySharkExample.
