Simple MacOS/Darwin Makefile

A very basic MacOS/Darwin Makefile for building a program using Shark with g++ looks like this: 

SHARKHOME=/opt/Shark/

LDLIBS=-lshark
LDFLAGS=-L${SHARKHOME}/lib
CXXFLAGS=-O3 -I${SHARKHOME}/include

The variable SHARKHOME must be set to where Shark was installed. Now, to compile the program MySharkExample.cpp just type make MySharkExample.
