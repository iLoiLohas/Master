Debugging and Exceptions

Running the following faulty program leads to undefined behavior: 

#include <Array/ArrayOp.h>

int main(int argc, char** argv)
{
    Array<int> m(3,2);

    std::cout << m(2, 3) << std::endl;

    return EXIT_SUCCESS;
}

However, if compiled with the option -DDEBUG, it will throw an exception, which can be handled in the following way: 

#include <Array/ArrayOp.h>

int main(int argc, char** argv)
{
    Array<int> m(3,2);

    try {
      std::cout << m(2, 3) << std::endl;
    } catch (const SharkException& e) {
      std::cout << e.what() << std::endl;
    }

    return EXIT_SUCCESS;
}
