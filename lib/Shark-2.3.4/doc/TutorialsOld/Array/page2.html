Streaming Arrays

There are two ways to write Arrays to an output stream. 

#include <Array/ArrayIo.h>
#include <Array/ArrayOp.h>

int main(int argc, char** argv)
{
    Array<double> a, b, c;
    a.resize(3u);
    b.resize(3u);

    a = 1.;

    std::cout << a << std::endl;
    writeArray(a, std::cout);											

    return EXIT_SUCCESS;
}
Array(3) 1 1 1 1 1 1

Note that Array/ArrayIo.h is required for IO-functions such as writeArray and Array/ArrayOp.h for general operations on Arrays, including basic numerical operators as well as stream operators.

Reading from streams is handled analogously, using the function readArray or the stream operator.
