Training Data

The first thing we will do is generate some sample data. In a real application data would usually be loaded from the hard disk, but here we will just sample our toy training examples from a fixed distribution called the "chess board". We use a dataset object to hold the corresponding arrays. 

#include <ReClaM/ArtificialDistributions.h>
#include <ReClaM/Dataset.h>

int main(int argc, char** argv)
{
    Chessboard chess;
    Dataset dataset(chess, 500, 10000);

    return EXIT_SUCCESS;
}

First, we construct a Chessboard object representing the distribution. Then we construct a dataset consisting of 500 training and 10000 test examples. The Dataset class simply holds four arrays: examples and labels for the training and the test set. We can access these arrays with the dataset members

  • getTrainingTarget()
