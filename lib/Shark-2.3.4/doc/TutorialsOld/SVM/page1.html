Introduction

This tutorial explains how to ...

  • train a Support Vector Machine (SVM)
  • classify data with the SVM

The tutorial is designed for users of the Shark library who want to use Support Vector Machines in their own programs. As Shark is a C++ library, it is necessary that you feel comfortable with the C++ language. This tutorial requires no experience in programming with Shark. Of course it is helpful it you are already familiar with the structure of the ReClaM module and with the Array library.

It is assumed that you are familiar with the concept of a support vector machine, at least at a user level. In particular, you should know about labeled training data, regularization, and kernel functions.
